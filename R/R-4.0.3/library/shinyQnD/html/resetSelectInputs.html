resetSelectionInputs {shinyQnD}R Documentation

Resetting SelectInputs to All

Description

Takes a collection of selectInput id's and sets them to "All". Useful in combination with with shiny::actionButton. Uses shiny::updateVarSelectInput

Usage

resetSelectionInputs(input, ids)

Arguments

session

the session variable from a shiny application

ids

a collection of selectInput ids to set to "All"

Author(s)

Bryen Moberg

See Also

generateChoices generateSelectInput

Examples

library(shiny)
 ui <- fluidPage(actionButton("reset","Reset")
 selectInput("COLOR", "Color", choices = c("All", "Other")))

 server <- function(input, output, session){

 resetSelections <- observeEvent(input$reset,{
 resetSelectInputs(c("COLOR"))
 })
}
shinyApp(server = server, ui  = ui)



## The function is currently defined as
resetSelectInputs <- function(session,inputNames){
  for(name in inputNames){
    updateVarSelectInput(session, name, selected = "All")
  }
}
